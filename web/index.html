KID MODE IS THE DEFAULT

Big pictures. Tiny decisions.

Setup controls disappear. Children choose a large cover tile, press Play, hear a game name when Windows speech is available, or open a simple offline help card.

  • Arrow keys choose; Enter plays
  • Space says the game name
  • H opens a parent-written help.txt
  • No store, account, chat or delete buttons
Kid Mode simplifies navigation; it is not a replacement for Windows parental controls or grown-up supervision.

Why not literally every console?

Newer systems require far more CPU/GPU power, different operating systems, BIOS files and emulator builds. Plugcade’s adapter format can grow without pretending a Pentium can emulate a PlayStation 2.
