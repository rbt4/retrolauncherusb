THE 20-SECOND ANSWER

Plugcade is the glue.

It is not a game store, a ROM downloader or one giant emulator bundle. It is the tiny layer that turns a messy personal collection into one portable library.

THE REAL MAGIC

More than a pretty launcher.

The important part is the portable system underneath the screen.

01

Many formats. One shelf.

Plugcade understands system-specific ROMs, zipped arcade sets, DOS folders and common disc-image formats instead of forcing everything into one file type.

Multi-file disc sets stay together; ambiguous files are not guessed silently.
02

Move it without breaking it.

Internal paths stay relative to the Plugcade folder. Put it on E: today and G: tomorrow—the portable library still knows where it lives.

Collections intentionally linked outside Plugcade are marked non-portable.
03

Your progress can travel too.

Every title gets a stable Saves\SYSTEM\GAME-ID location. Core 0.4 can also make one dated save backup per day and keeps the newest three.

Honest boundary: Plugcade exposes the save path, but cannot force every emulator to support portable saves or timed savestates.
04

One library. Different runners.

The folder layout stays familiar while each edition can use a launcher and emulator build appropriate to Windows 11, Windows 7, XP, Windows 98—or eventually native DOS.

Old hardware gets frozen, tested tools instead of a huge modern runtime.
HOW IT ACTUALLY WORKS

Grown-up setup once. Easy play afterwards.

Plugcade is portable, but an emulator still has to match the computer and console. The alpha keeps that setup visible instead of pretending it is automatic.

  1. 1
    Unzip PlugcadePut the whole folder on a USB drive or directly on the computer. Nothing installs into Windows.
  2. 2
    Choose your systemsPlugcade creates matching Drop, Library, Emulator and Save folders only for the systems you enable.
  3. 3
    Add games and emulatorsDrop files into the obvious folders or select an existing organized collection. Import copies safely; originals remain.
  4. 4
    Open Kid ModeChildren see large covers and Play, Say and Help—not setup tools, file browsers or delete buttons.
Why are emulators separate?A Windows 98 PC, an XP laptop and a Windows 11 desktop need different emulator builds. Keeping them modular is what lets Plugcade stay tiny and reach much older hardware.

WHAT CAN LIVE IN THE LIBRARY?

Old games arrive in different shapes.

Plugcade uses platform adapters so a DOS folder, an arcade ZIP and a PlayStation disc image can appear together without being treated as the same thing.

ROM FILES

One-file games

Examples include .nes, .sfc, .gb, .gba, .z64, .md, .pce and many more.

DISC GAMES

Images and track sets

Plugcade recognizes formats such as .iso, .chd and descriptor-led sets such as .cue + .bin.

COMPUTER GAMES

Complete DOS folders

Keep the whole game folder together. Plugcade looks for a suitable .exe, .com or .bat launch target.

An adapter is not an emulator.

It gives Plugcade the right folder, file types, save location and launch recipe. The chosen emulator, BIOS, operating system and hardware still determine whether a game runs well.

ONE LIBRARY, MULTIPLE COMPUTER AGES

“Old PC” is not one target.

A Windows 11 desktop and a Windows 98 tower should not receive the same enormous package. Plugcade’s shared library structure stays familiar while each edition freezes the correct launcher and emulator generation.
