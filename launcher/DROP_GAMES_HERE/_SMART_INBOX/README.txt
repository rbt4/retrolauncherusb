PLUGCADE SMART INBOX
====================

Drop clearly identifiable game files here, then start Plugcade or click
IMPORT NOW. Examples: .nes, .gba, .z64, .sfc, .gbc and .pce.

Ambiguous files such as .bin, .rom, .iso and .cue must go in a named system
folder instead. Plugcade reports them rather than guessing incorrectly.

Original files are never moved, deleted or overwritten.
